# 디버깅 도구를 먼저 구축한다

AI 애플리케이션 개발 시 디버깅·관찰 도구를 반드시 먼저 구현해놓고 진행한다.

## 대상 도구 예시

- Burr UI
- LangSmith
- LangWatch
- 그 외 트레이싱·로깅·상태 관찰 도구

## 이유

AI 애플리케이션은 내부 추론 과정이 불투명하다. 디버깅 도구 없이는:
- 어느 단계에서 문제가 발생했는지 특정할 수 없다.
- 프롬프트·도구 호출·중간 출력을 추적할 수 없다.
- 개선의 효과를 정량적으로 확인할 수 없다.

관찰 체계가 먼저 갖춰져야 개발 속도가 유지되고, 문제 발생 시 원인 파악이 가능하다.
