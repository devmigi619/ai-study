# 리뷰에이전트는 필수이다

코딩에이전트 팀에는 반드시 리뷰어 / 리뷰 전문가 / 리뷰 에이전트가 포함되어야 한다.

## 역할

- 작업 에이전트가 생성한 코드·문서·결정을 독립적으로 검토한다.
- 이 지식 베이스의 원칙 문서들을 검토 기준으로 삼는다.
- 승인 없이 본선 반영되지 않도록 게이트 역할을 수행한다.

## 리뷰 에이전트가 참고해야 할 원칙

- 본 지식 베이스의 모든 원칙 문서
- 프로젝트별 작업 컨텍스트 (예: CLAUDE.md의 영역 침범 방지 규칙)

## 이유

에이전트는 자신의 실수를 스스로 인지하지 못하는 경우가 많다. 독립적인 리뷰 레이어가 버그·원칙 위반·영역 침범을 걸러내야 전체 품질이 유지된다.
