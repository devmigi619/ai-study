# 쉬운 말로 쓴다

코딩에이전트가 만드는 문서와 코드는 쉬운 말로 쓴다. "정말 제대로 아는 사람은 쉽게 설명할 수 있다."

## 이유

- 전문 용어는 아는 사람끼리만 빠르게 소통하게 만든다. 맥락 모르는 사람은 문 앞에서 막힌다.
- 어려운 말을 쓰는 건 이해가 얕다는 신호일 때가 많다. 깊이 아는 사람은 일상 말로 바꿀 수 있다.
- 지식 문서는 다음 사람이 읽을 때 빛난다. 그 사람이 처음 읽는 사람이라면?

## 적용

- 전문 용어는 쓸 때 바로 풀어쓴다. "재현율(관련 결과를 얼마나 많이 찾아냈는가)"처럼.
- 꼭 써야 하는 영어 용어는 본문에 한 번 한국어 뜻을 적어둔다.
- 문장은 짧게. 한 문장에 한 가지.
- "정말 제대로 아는 사람은 쉽게 설명할 수 있다"를 잣대로 삼는다. 어려운 말이 튀어나오면 멈추고 다시 쓴다.
- 사용자가 요청/언급하면 전문 용어를 사용해도 된다.
- 전문 용어가 꼭 필요하면 사용해도 된다.
